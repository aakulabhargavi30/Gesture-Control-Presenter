# Gesture Control Presenter | Computer Vision & HCI 

Domain: Computer Vision, Machine Learning, Human-Computer Interaction  
Status: Real-time gesture recognition achieving 95% accuracy at 30 FPS

Control Google Slides presentations touchlessly using hand gestures. 
This project implements a real-time Computer Vision pipeline using MediaPipe's pre-trained hand landmark detection model to enable Human-Computer Interaction via webcam.

---

#  Gesture Controls 🧠

| Gesture          | Action          |
| ---------------- | --------------- |
| ✋ Open Palm      | Next Slide      |
| 🤟 Three Fingers | Previous Slide  |
| ✌️ Two Fingers   | Start Slideshow |
| ✊ Fist           | Exit Slideshow  |
